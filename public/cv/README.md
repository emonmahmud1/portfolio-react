# CV Files

Place your CV files here:

1. `developer-cv.pdf` - Your Full-Stack Developer CV
2. `sqa-cv.pdf` - Your SQA Engineer CV

## How to add your CVs:

1. Rename your CV files to match the names above
2. Copy them to this folder (public/cv/)
3. The download buttons will automatically work

## File Requirements:
- Format: PDF
- File names must match exactly (case-sensitive)
- Keep file size reasonable (< 5MB recommended)
